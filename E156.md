# E156 Micro-Paper: Global Transportability Atlas

**Project:** Global Transportability Assessment of Cardiovascular/Diabetes Prognostic Models using GBD 2023 Covariates
**Protocol Date:** 2026-04-10

**CURRENT BODY:**
Can cardiovascular and diabetes prognostic models maintain predictive accuracy when transported from high-income countries to the Global South using contemporary GBD 2023 covariates? We extracted 1980-2023 covariates from the Global Burden of Disease study, including socioeconomic, health system access, and demographic variables for 204 countries. Our framework applies a "TruthCert" pipeline using 2026-standard Double Machine Learning (DML), assesses model brittleness via Fisher Information Stability on the statistical manifold, and quantifies manifold mismatch using Topological Bottleneck Distance (TDA). Transportability drift is fairly decomposed using Shapley Value Attribution to identify the specific covariate drivers of recalibration needs. Preliminary mapping in four regions (IND, NGA, KEN, BRA) shows that stability scores below 40% correlate with catastrophic calibration failure, while TDA bottleneck distances identify population clusters structurally invisible to Western-centric models. Robustness checks using Anytime-Valid E-statistics and conformal prediction confirmed the sequential stability of our evidence base. These results provide a "Global Transportability Atlas" for clinicians, identifying high-risk regions where non-linear causal recalibration is mandatory. The study is limited by the assumptions of nuisance parameter orthogonality and requires high-fidelity topological descriptors.
